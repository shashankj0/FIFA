# Security Policy

## Supported Versions

We actively provide security support and updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

If you discover a potential security vulnerability in ArenaMind 2026, please do **NOT** open a public issue. Instead, report it privately:

1. Send an email describing the vulnerability to **security@arenamind2026.local**.
2. Include a detailed description, reproduction steps, and a proof of concept (PoC) if available.
3. Our team will acknowledge receipt of the report within 48 hours and work with you to coordinate a fix and disclosure.
